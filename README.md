# Coffee-shop-pos
Excel VBA Point of Sale system for managing coffee shop orders, products and sales transactions.

<img width="1113" height="535" alt="coffee" src="https://github.com/user-attachments/assets/45524496-d435-4caf-8337-3db5cd18f09b" />

## Features

- Point of Sale (POS) interface built with Excel VBA
- User-friendly order entry through a VBA UserForm interface
- Product selection with automatic price calculation
- Automatic order total calculation
- Sales data storage in structured Excel tables
- Quick order processing designed for small coffee shop environments
- Simple and efficient interface for fast transaction handling


## Technologies Used

- Microsoft Excel
- VBA (Visual Basic for Applications)
- Excel Tables
- UserForms (VBA UI)
- Basic sales data management
